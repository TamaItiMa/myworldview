Ideas for Apps

  1. Enter two numbers use operators on them + - * / print the results %
  2. Create a password based on a name
  3. Check if name is on a list
  4. Write a program that prints ‘Hello World’ to the screen.
  5. Write a program that asks the user for their name and greets them with their name.
  6. Modify the previous program such that only the users Alice and Bob are greeted with their names.
  7. Write a program that asks the user for a number n and prints the sum of the numbers 1 to n
  8. Modify the previous program such that only multiples of three or five are considered in the sum, e.g. 3, 5, 6, 9, 10, 12, 15 for n=17
  9. Write a program that prints a multiplication table for numbers up to 12.
  10. Write a program that prints all prime numbers. (Note: if your programming language does not support arbitrary size numbers, printing all primes up to the largest number you can easily represent is fine too.)
  11. Write a program that prints all prime numbers. (Note: if your programming language does not support arbitrary size numbers, printing all primes up to the largest number you can easily represent is fine too.)
  12. Write a guessing game where the user has to guess a secret number. After every guess the program tells the user whether their number was too large or too small. At the end the number of tries needed should be printed. It counts only as one try if they input the same number multiple times consecutively.
